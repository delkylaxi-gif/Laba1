# Решение диофантова уравнения ax + by = c
## Описание
Программа находит целочисленное решение линейного диофантова уравнения вида ax + by = c с минимальным неотрицательным значением x.
## Алгоритм
Программа использует расширенный алгоритм Евклида для поиска решения уравнения, после чего приводит его к общему виду и находит минемальное неотрицательное значение x.

### Шаги работы:
1. Проверка существования решения — уравнение имеет решение только если c делится на НОД(a, b)
2. Поиск частного решения (x₀, y₀) для уравнения ax + by = НОД(a, b)
3. Масштабирование — умножение коэффициентов на c/НОД(a, b)
4. Нахождение минимального x — приведение к минимальному неотрицательному значению по модулю

## Входные данные
Три целых числа a b c, разделённые пробелами:
- a — коэффициент при x
- b — коэффициент при y  
- c — свободный член

## Выходные данные
- x y — минимальное неотрицательное решение, если оно существует
- Impossible — если уравнение не имеет целочисленных решений
## Пример использования

```bash
$ python3 solution.py
6 9 3
2 -1
bash
Copy
$ python3 solution.py
2 4 3
Impossible
```
Функции
Nod(a, b)
Рекурсивная реализация расширенного алгоритма Евклида.
Параметры:
a, b — целые числа
Возвращает:
gcd — наибольший общий делитель
x, y — коэффициенты Безу, такие что ax + by = gcd
Сложность алгоритма
Время: O(log(min(a, b)))
Память: O(log(min(a, b))) — глубина рекурсии
